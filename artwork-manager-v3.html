セットアップ手順：
1. Apps Script コード をコピーして、Google スプレッドシートのスクリプトエディタに貼り付けます
2. 「デプロイ」→「新しいデプロイ」→ 種類：ウェブアプリ → 実行ユーザー：自分 → アクセスできるユーザー：全員 → デプロイ
3. 初回実行後に作成される config シートの apiToken に長めのランダム文字列を設定します
4. 表示された URL と APIトークンをここに貼り付けて「接続・読み込み」を押します
※ データ用の artwork と更新履歴用の audit_log シートも自動作成されます
